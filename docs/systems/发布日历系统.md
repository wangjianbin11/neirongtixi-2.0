## 发布调度规则

### 自动调度规则

```yaml
规则1: 关键内容优先
  触发: S级话题创建
  动作: 自动分配到最佳时间段
  平台: YouTube + SEO文章

规则2: 内容复用
  触发: YouTube长视频发布
  动作: 自动创建短视频任务
  平台: TikTok + Shorts + Reels

规则3: 社交媒体发布
  触发: SEO文章发布
  动作: 自动创建社媒分享任务
  平台: Twitter + LinkedIn + Facebook

规则4: 工作日/周末调整
  触发: 时间检查
  动作: 自动调整发布时间
  规则: 工作日发布专业内容，周末发布轻松内容
```

### 冲突检测

```yaml
时间冲突:
  - 检测: 同一平台相近时间发布
  - 提示: 警告用户
  - 建议: 建议间隔至少2小时

资源冲突:
  - 检测: 同一创作者任务重叠
  - 提示: 资源冲突提醒
  - 建议: 调整任务时间或分配其他人员

内容冲突:
  - 检测: 相似主题内容相近时间发布
  - 提示: 内容重复提醒
  - 建议: 错开发布或整合内容
```

## 工作流集成

### 内容生产流程

```yaml
步骤1: 话题创建
  → 添加到发布日历
  → 状态: 待制作

步骤2: 内容创作
  → 更新状态: 制作中
  → 预计完成时间

步骤3: 内容审核
  → 更新状态: 待审核
  → 审核不通过: 返回修改

步骤4: 计划发布
  → 设置发布时间
  → 状态: 计划发布

步骤5: 自动发布
  → 到达发布时间
  → 自动执行发布
  → 状态: 已发布

步骤6: 发布后跟进
  → 监控数据
  → 回复评论
  → 记录表现
```

### 批量发布

```yaml
场景: 营销活动/产品发布
功能:
  - 批量创建发布任务
  - 批量设置发布时间
  - 批量更新状态
  - 批量导入导出

注意:
  - 确保内容质量
  - 避免平台垃圾信息判定
  - 保持自然发布节奏
```

## 提醒和通知

### 提醒类型

```yaml
发布前提醒:
  - 时间: 发布前1小时
  - 内容: 发布内容摘要
  - 动作: 确认或取消

发布提醒:
  - 时间: 到达发布时间
  - 内容: 开始发布
  - 动作: 查看进度

审核提醒:
  - 时间: 内容提交审核
  - 内容: 待审核内容
  - 动作: 立即审核

延期提醒:
  - 时间: 未按计划发布
  - 内容: 延期内容
  - 动作: 重新安排

数据报告:
  - 时间: 发布后24小时
  - 内容: 表现数据
  - 动作: 查看报告
```
