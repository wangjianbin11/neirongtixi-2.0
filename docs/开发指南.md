# ASG内容系统 - 开发指南

## 开发环境设置

### 前置要求

- Node.js 20+
- PostgreSQL 15+
- MongoDB 7+
- Redis 7+
- Docker & Docker Compose (可选)

### 快速启动

1. **安装所有依赖**
```bash
npm run install:all
```

2. **启动数据库**
```bash
npm run docker:up
```

3. **初始化数据库**
```bash
npm run db:migrate
```

4. **启动开发服务器**
```bash
npm run dev
```

访问:
- 前端: http://localhost:3000
- 后端: http://localhost:4000

## 开发规范

### 前端开发

1. **组件命名**
   - PascalCase: 组件文件 (`UserList.tsx`)
   - camelCase: 工具函数 (`formatDate.ts`)

2. **状态管理**
   - 全局状态使用 Zustand
   - 局部状态使用 useState

3. **样式**
   - 使用 Ant Design 组件
   - 内联样式用于动态样式
   - 避免使用 CSS-in-JS

4. **API调用**
   - 使用对应的API客户端
   - 添加错误处理
   - 显示加载状态

### 后端开发

1. **路由结构**
   ```
   GET /api/v1/resources       - 列表
   GET /api/v1/resources/:id   - 详情
   POST /api/v1/resources      - 创建
   PUT /api/v1/resources/:id   - 更新
   DELETE /api/v1/resources/:id - 删除
   ```

2. **错误处理**
   - 使用 try-catch
   - 返回统一格式:
   ```json
   {
     "success": false,
     "error": {
       "code": "ERROR_CODE",
       "message": "Error message"
     }
   }
   ```

3. **认证**
   - 使用 `authenticate` 中间件
   - 通过 `req.userId` 获取用户ID

## 添加新功能

### 添加新页面

1. 创建页面组件:
```typescript
// frontend/src/pages/newpage/NewPage.tsx
export default function NewPage() {
  return <div>New Page</div>;
}
```

2. 添加路由 (`App.tsx`):
```typescript
import NewPage from './pages/newpage/NewPage';

<Route path="newpage" element={<NewPage />} />
```

3. 添加菜单项 (`Layout.tsx`):
```typescript
{
  key: '/newpage',
  icon: <IconOutlined />,
  label: '新页面',
}
```

### 添加新API

1. 创建服务 (`backend/src/services/`):
```typescript
export class NewService {
  async method() {
    // 实现
  }
}
export const newService = new NewService();
```

2. 创建路由 (`backend/src/api/`):
```typescript
import { newService } from '../services/newService';

export const newRouter = Router();

newRouter.get('/', authenticate, async (req, res) => {
  // 实现
});
```

3. 注册路由 (`backend/src/api/index.ts`):
```typescript
import { newRouter } from './new';
apiRouter.use('/new', newRouter);
```

4. 创建前端API客户端:
```typescript
export const newApi = {
  method: async () => {
    return await apiClient.get('/new');
  };
};
```

## 数据库操作

### 查询数据
```typescript
const result = await pool.query('SELECT * FROM table WHERE id = $1', [id]);
return result.rows[0];
```

### 插入数据
```typescript
const result = await pool.query(
  'INSERT INTO table (col1, col2) VALUES ($1, $2) RETURNING *',
  [value1, value2]
);
return result.rows[0];
```

### 更新数据
```typescript
const result = await pool.query(
  'UPDATE table SET col1 = $1 WHERE id = $2 RETURNING *',
  [newValue, id]
);
return result.rows[0];
```

### 删除数据
```typescript
await pool.query('DELETE FROM table WHERE id = $1', [id]);
```

## 常见问题

### 端口冲突
- 前端默认 3000
- 后端默认 4000
- 在 `.env` 文件中修改

### 数据库连接失败
1. 检查 PostgreSQL 是否运行
2. 验证 `DATABASE_URL` 配置
3. 确认数据库已创建

### CORS 错误
检查 `backend/.env` 中的 `CORS_ORIGIN` 配置

### 认证失败
1. 检查 JWT_SECRET 配置
2. 确认 token 未过期
3. 清除浏览器 localStorage
