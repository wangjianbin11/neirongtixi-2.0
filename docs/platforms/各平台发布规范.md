## YouTube发布规范

### 长视频规范

#### 内容要求
- **时长**: 10-30分钟（最佳15-20分钟）
- **质量**: 最低1080p，推荐4K
- **格式**: 16:9横屏
- **字幕**: 必须提供SRT字幕文件
- **缩略图**: 1280x720，<2MB

#### SEO要求
```yaml
标题:
  - 长度: 50-60字符
  - 包含: 主关键词
  - 风格: 吸引点击

描述:
  - 前150字: 核心描述，包含关键词
  - 结构: 完整内容大纲
  - 链接: 相关链接和CTA
  - 章节: 时间戳章节标记

标签:
  - 主标签: 核心关键词
  - 次标签: 相关关键词5-10个
  - 品牌标签: ASG相关
```

#### 发布流程
```
1. 内容创作 → 2. 视频制作 → 3. SEO优化 → 4. 上传视频
→ 5. 添加字幕 → 6. 发布时间设置 → 7. 发布后推广
```

#### 发布时间
- **最佳**: 周六下午2-4点
- **次优**: 周日下午2-4点
- **避免**: 周一至周五上午9-10点（工作时间）

### Shorts规范

#### 内容要求
- **时长**: 15-60秒（最佳30-45秒）
- **格式**: 9:16竖屏
- **字幕**: 必须配字幕
- **音乐**: 使用热门音乐

#### 发布要求
```yaml
标题:
  - 简短有力
  - 包含关键词
  - 吸引点击

描述:
  - 100-200字
  - 包含关键词
  - 话题标签

话题标签:
  - #Shorts
  - 3-5个相关标签
```
